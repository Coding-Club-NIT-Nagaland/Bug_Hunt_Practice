# Question 2: Find the Maximum

## Problem Statement

Write a function that finds the **maximum** value in an array of integers.

## Example

* `numbers` = {1, 5, 2, 9, 4}
* **Expected Output:** `9`

* `numbers` = {-10, -5, -3, -8}
* **Expected Output:** `-3`

The buggy code works for the first example but fails for the second. Find the bug!
